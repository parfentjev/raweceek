# Rawe Ceek Frontend Application

This is a web application for the backend service developed in this project.

You can see it in action here: [https://raweceek.eu/](https://raweceek.eu/)

## Getting Started

To build and run the application, follow these steps:

-   Create a new `.env` file based on the `example.env` file.
-   Build static files: `npm build`

Also, see the `Makefile`.
